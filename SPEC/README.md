# AiOS Target Specifications

This directory owns AiOS target specifications only.

Do not define AiLang language semantics, AiVM evaluation semantics, or AiVectra
UI semantics here. AiOS specs may define:

- base image layout
- boot/profile contracts
- target option behavior
- QEMU/device runner behavior
- host library packaging requirements
- AiOS-specific display/input backend selection
